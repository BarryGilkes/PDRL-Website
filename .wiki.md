# Project Summary
The Pro Drag Racing League (PDRL) is an interactive platform designed for drag racing enthusiasts, aimed at enhancing their racing experience through features such as event registration, performance tracking, and community engagement. It facilitates event management and user interaction using a modern technology stack that includes React, TypeScript, and Tailwind CSS for the frontend, and FastAPI with PostgreSQL for the backend, ensuring a seamless user experience and efficient data management.

# Project Module Description
The project comprises the following functional modules:
- **Frontend**: A React application using TypeScript and Tailwind CSS, providing user-friendly interfaces for event listings, registrations, leaderboards, and an admin panel.
- **Backend**: A FastAPI application that manages data, user authentication, and provides API endpoints for event registrations, race times, and driver profiles.
- **Database**: PostgreSQL is used to store event details, registration information, race times, and user data.

# Usage
1. **Install Dependencies**:
   - For the backend, run `pip install -r requirements.txt`.
   - For the frontend, run `pnpm install`.

2. **Build the Project**:
   - Navigate to the frontend directory and run `pnpm run build`.

3. **Run the Application**:
   - Start the backend server using `uvicorn app.backend.main:app --reload`.

4. **Import Race Data**:
   - Use the admin panel to upload race data.

5. **View Event Flyers**:
   - Access event flyers and details through the public-facing pages, now with lightbox functionality for larger views.

6. **Health Check Endpoint**:
   - Verify the backend's operational status via the health check endpoint.
